# kick-scooter-discount-frontend

Frontend for the kick-scooter-discount microservice.
Built with [Next.js](https://nextjs.org).

## Dependencies
 - NodeJS 12

## How to run in local
 - Clone respository
 - `npm install`
 - Change backend URL in `apiClient.js`
 - `npm run dev`
 - Open http://localhost:3000
